AnsMath#

class ansys.math.core.math.AnsMath(mapdl=None, **kwargs)#

Provides the common class for abstract math objects.

Examples

Create an instance.

>>> import ansys.math.core.math as pymath
>>> mm = pymath.AnsMath()

Add vectors.

>>> v1 = mm.ones(10)
>>> v2 = mm.ones(10)
>>> v3 = v1 + v2

Multiply matrices (not yet available).

>>> v1 = mm.ones(10)
>>> m1 = mm.rand(10, 10)
>>> v2 = m1*v1
