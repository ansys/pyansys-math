User guide
==========

Overview
--------

You can use the AnsMath() method to launch an instance of PyAnsys Math.

.. code:: python

    import ansys.math.core.math as pymath

    # Start PyAnsys Math.
    mm = pymath.AnsMath()
    print(mm)
